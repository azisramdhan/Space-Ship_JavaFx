Game Space Shooter ini dibuat untuk memenuhi tugas masa depan (tugas besar) saya dalam mata kuliah pemrograman berorientasi objek yang diampu oleh dosen saya Ibu Rosa Ariani Sukamto, terimakasih sebesar-besarnya kepada beliau yang telah memberikan saya kesempatan untuk membuat game ini. Game ini dibuat dengan menggunakan bahasa Java dan framework Javafx untuk membuat User Interface nya.
